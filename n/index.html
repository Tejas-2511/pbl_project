PortionVision

"An intelligent Progressive Web Application that generates personalized portion recommendations and estimates food intake using OCR and future depth-based image analysis."

Problem Statement

In hostel mess environments, students self-serve food without structured guidance on appropriate portion sizes based on their individual nutritional requirements. Differences in age, height, weight, activity level, and health goals often result in nutrient imbalance. Manual calorie logging is subjective and inconsistent, and estimating portion size from a single RGB image is difficult due to the absence of explicit depth information.

Literature Insight

Depth estimation models (Depth Anything V2, MiDaS), segmentation models (SAM, SAM 2.0), and IFCT 2017 nutrition database form the academic foundation of this work.

Research Gap

Existing calorie apps rely on manual input and lack automated menu digitization and 3D portion estimation tailored for Indian hostel environments.

System Methodology

Menu OCR Pipeline

Image Upload → Sharp Preprocessing → Tesseract.js OCR → Cleaning & Deduplication → JSON Database Update.

Personalized Nutrition Engine

User Profile Input → BMI & BMR Calculation → Daily Calorie & Protein Requirement → Planned Portion Recommendation Logic.

PWA Implementation

React + Vite Frontend, Express Backend, Secure File Handling, Offline Caching via Service Worker.

Results & Analysis

Current Status OCR & PWA Implemented

The system successfully implements secure OCR-based menu extraction, structured food database management, personalized nutritional computation, and Progressive Web App deployment. Depth-based portion estimation and geometric volume analysis are currently under development.

Future Enhancement: RGB Image to Portion Size Estimation

  • Monocular depth estimation for height extraction
  • Food segmentation using SAM/SAM 2.0
  • Plate edge detection and scale calibration
  • 3D volume integration and mass estimation
  • Nutrition mapping using IFCT 2017
  • Actual vs recommended portion comparison with feedback
